Machine Learning Pipeline

Automated ML pipeline for customer segmentation and predictive analytics with real-time model deployment and monitoring.

This project, built by Mikkel Fan, implements a comprehensive Machine Learning Pipeline for automated customer segmentation and predictive analytics using Python, Scikit-learn, and cloud deployment technologies.

Problem

Manual customer segmentation processes were time-consuming and inconsistent, leading to suboptimal marketing campaigns and customer retention strategies. The business needed an automated solution for real-time customer analysis and predictive modeling.

Approach

  • Implemented K-means clustering and RFM analysis for customer segmentation using Python and Scikit-learn.
  • Built automated data pipeline with ETL processes, feature engineering, and model training workflows.
  • Deployed real-time prediction API using Flask and cloud infrastructure for instant customer scoring.
  • Integrated model monitoring and A/B testing frameworks for continuous improvement.

from sklearn.cluster import KMeans
from sklearn.preprocessing import StandardScaler
import pandas as pd

def segment_customers(df):
    # RFM Analysis
    rfm = df.groupby('customer_id').agg({
        'date': lambda x: (df['date'].max() - x.max()).days,
        'amount': ['sum', 'count']
    }).round(1)
    
    # K-means Clustering
    scaler = StandardScaler()
    features = scaler.fit_transform(rfm)
    kmeans = KMeans(n_clusters=4, random_state=42)
