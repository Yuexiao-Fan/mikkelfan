Twitter Analysis

Social media sentiment analysis and trend prediction using natural language processing, machine learning, and real-time data processing techniques.

This project, developed by Mikkel Fan, focuses on social media sentiment analysis and trend prediction using Twitter data, implementing advanced NLP techniques and machine learning models for real-time insights.

Problem

Businesses and organizations needed to understand public sentiment, track brand mentions, and predict social media trends in real-time to make informed decisions and respond to emerging issues quickly and effectively.

Approach

  • Implemented Twitter API integration for real-time data collection and streaming of tweets with relevant keywords and hashtags.
  • Developed sentiment analysis models using VADER, TextBlob, and custom BERT-based classifiers for accurate emotion detection.
  • Built trend prediction algorithms using time series analysis and machine learning to forecast viral content and emerging topics.
  • Created interactive dashboards with real-time visualizations of sentiment trends, topic clusters, and influence networks.

# Twitter Sentiment Analysis
import tweepy
import pandas as pd
from textblob import TextBlob
from vaderSentiment.vaderSentiment import SentimentIntensityAnalyzer
import matplotlib.pyplot as plt

class TwitterAnalyzer:
    def __init__(self, api_key, api_secret):
        self.api = self.setup_twitter_api(api_key, api_secret)
        self.sia = SentimentIntensityAnalyzer()
    
    def collect_tweets(self, query, count=1000):
        tweets = tweepy.Cursor(
            self.api.search_tweets,
            q=query,
            lang="en",
            tweet_mode="extended"
        ).items(count)
        
        return [tweet.full_text for tweet in tweets]
    
    def analyze_sentiment(self, tweets):
        sentiments = []
        for tweet in tweets:
            # VADER sentiment
            vader_score = self.sia.polarity_scores(tweet)
            
            # TextBlob sentiment
            blob = TextBlob(tweet)
            textblob_score = blob.sentiment.polarity
            
            # Combined sentiment
            combined_score = (vader_score['compound'] + textblob_score) / 2
            sentiments.append(combined_score)
        
        return sentiments
    
    def predict_trends(self, sentiment_data):
        # Time series analysis for trend prediction
        from sklearn.linear_model import LinearRegression
        import numpy as np
        
        X = np.arange(len(sentiment_data)).reshape(-1, 1)
        y = sentiment_data
        
        model = LinearRegression()
        model.fit(X, y)
        
        # Predict next 24 hours
        future_X = np.arange(len(sentiment_data), len(sentiment_data) + 24).reshape(-1, 1)
        predictions = model.predict(future_X)
        
        return predictions
