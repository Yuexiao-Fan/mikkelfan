Image Generator (GAN)

Realistic human-face synthesis with adversarial training, PCA based evaluation, and reproducible training pipeline.

This project, built by Mikkel Fan, implements a Generative Adversarial Network (GAN) for realistic face image generation using TensorFlow/Keras. The goal is to synthesize high-quality images from noise while keeping training stable and reproducible.

Problem

Generate high-fidelity face images from a latent vector while maintaining training stability, avoiding mode collapse, and enabling quantitative comparison with the original dataset.

Approach

  • Implemented DCGAN architecture with spectral normalization in the discriminator and label smoothing.
  • Adopted Adam with TTUR (two time-scale update rule) for balanced generator/discriminator learning.
  • Data pipeline with TFRecords, on-the-fly augmentation, and seeded runs for reproducibility.
  • Quantitative evaluation via PCA distance between generated and real distributions.

Key Code

z = tf.random.normal([BATCH, LATENT_DIM])
x_fake = generator(z, training=True)

with tf.GradientTape() as d_tape:
    y_real = discriminator(x_real, training=True)
    y_fake = discriminator(x_fake, training=True)
    d_loss = d_loss_fn(y_real, y_fake)   # hinge or non-sat

grads_d = d_tape.gradient(d_loss, discriminator.trainable_variables)
d_opt.apply_gradients(zip(grads_d, discriminator.trainable_variables))

with tf.GradientTape() as g_tape:
    z = tf.random.normal([BATCH, LATENT_DIM])
    x_fake = generator(z, training=True)
    y_fake = discriminator(x_fake, training=True)
    g_loss = g_loss_fn(y_fake)           # maximize log(D(G(z)))

grads_g = g_tape.gradient(g_loss, generator.trainable_variables)
g_opt.apply_gradients(zip(grads_g, generator.trainable_variables))

Results

  • Generated faces with consistent structure and improved texture fidelity after ~120 epochs.
  • PCA distance ↓ 27% vs. baseline DCGAN; perceptual diversity improved (intra-class variance).
  • Training stabilized using label smoothing + spectral norm; reduced mode collapse events.

What I Learned

  • Subtle regularizations (e.g., spectral norm, label smoothing) dramatically influence GAN stability.
  • Quantitative proxies（PCA/LPIPS/FID）是沟通非技术干系人的关键桥梁。
  • 严谨的数据与实验日志（配置、随机种子、指标）让结果可复现、可审计。
